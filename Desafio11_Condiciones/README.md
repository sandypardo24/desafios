# Desafio 11 Condicionales

Este repositorio contiene tres proyectos diferentes que demuestran el desafio 11 de JavaScript.

## Proyecto 1: Imagen con Borde Rojo

### Descripción
Este proyecto muestra una página con una única imagen. Al hacer clic en la imagen, se le agrega un borde rojo de 2 píxeles. Si ya tiene el borde, al hacer clic nuevamente se le quita el borde.

### Archivos

#### index1.html
#### styles1.html
#### script1.html


## Proyecto 2: Contador Stickers

### Descripción
Este proyecto permite seleccionar hasta 10 stickers de tres tipos disponibles. Muestra un mensaje indicando la cantidad de stickers seleccionados o si se ha excedido el límite.

### Archivos

#### index2.html
#### styles2.html
#### script2.html

## Proyecto 3: password Check

### Descripción
Este proyecto permite ingresar un password usando tres selectores. Dependiendo del valor ingresado, muestra un mensaje indicando si el password es correcto o incorrecto.

### Archivos

#### index3.html
#### styles3.html
#### script3.html
